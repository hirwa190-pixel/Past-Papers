# 🎓 Ultimate Study Hub

A modern educational web platform built to help students revise, practice, and learn easily through past papers, notes, quizzes, and more.

---

## 🚀 Features

- 📚 Access to Past Papers
- 📝 Self-Study Resources
- 🔍 Search Functionality
- 👤 User Authentication (Login & Signup)
- 📱 Responsive Design
- ⚡ Fast and Simple User Interface
- 📖 Educational Content for Students
- 🎯 Revision-Friendly Layout

---

## 🛠️ Technologies Used

- Vue.js
- HTML
- CSS
- JavaScript
